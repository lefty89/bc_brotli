Bugcluster Brotli
=============================================================

.. only:: html

	:Classification:
		bc_brotli

	:Version:
		|release|

	:Language:
		en

	:Description:
		Makes Googles new brotli compression available for TYPO3 JavaScript and CSS optimisation. Compared to gzip, brotli reduces the size of resources (html, css, js) between 17% and 25% and allows faster loading of websites.
